# 🎯 A-6 Digital Tools Buying Website

## 📌 Project Overview

This is a modern and responsive **Digital Tools Buying Website** where users can explore different digital products and purchase them easily. The project focuses on clean UI, smooth user experience, and functional cart management using React.

--

## 🚀 Live Demo

🔗 Live Site: *[https://dainty-basbousa-16436f.netlify.app/]*
🔗 GitHub Repo: *[https://github.com/najiba-ta/Digi-tools-buying]*

---

## 🛠️ Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS & DaisyUI
* 📜 JavaScript (ES6+)
* 🔔 React-Toastify
* 📦 JSON (for product data)

---

## ✨ Key Features

### 🛒 1. Smart Cart System

* Add products to cart
* Remove individual items
* Shows total product count in Navbar
* "Proceed to Checkout" clears the cart

### 🔄 2. Product & Cart Toggle

* Toggle between **Products** and **Cart section**
* Product section is visible by default
* Cart shows empty message initially

### 🔔 3. Toast Notifications

* Shows alert when:

  * Product added
  * Product removed
  * Checkout completed

## 🧩 Main Sections

## 📦 Product Data Structure

Each product includes:

* id
* name
* description
* price
* period (one-time / monthly / yearly)
* tag & tagType
* features (array)
* icon
